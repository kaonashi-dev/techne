title: "401 Unauthorized"
description: "RFC 7807 problem document for HTTP 401 Unauthorized."
---

This page documents the Techne RFC 7807 problem document for HTTP `unauthorized` errors.

A `problem+json` response with `type` of
`https://github.com/kaonashi-dev/techne/blob/main/docs/errors/unauthorized.md`
indicates the server encountered an Unauthorized condition.

```json
{
  "type": "https://github.com/kaonashi-dev/techne/blob/main/docs/errors/unauthorized.md",
  "title": "Unauthorized",
  "status": 401,
  "detail": "<human-readable explanation>",
  "code": "<optional stable machine-readable code>",
  "instance": "<request URL path>",
  "requestId": "<uuid>"
}
```

For the full error contract and `HttpException` reference, see [Exceptions](/http/exceptions).

title: "404 Not Found"
description: "RFC 7807 problem document for HTTP 404 Not Found."
---

This page documents the Techne RFC 7807 problem document for HTTP `not-found` errors.

A `problem+json` response with `type` of
`https://github.com/kaonashi-dev/techne/blob/main/docs/errors/not-found.md`
indicates the server encountered a Not Found condition.

```json
{
  "type": "https://github.com/kaonashi-dev/techne/blob/main/docs/errors/not-found.md",
  "title": "Not Found",
  "status": 404,
  "detail": "<human-readable explanation>",
  "code": "<optional stable machine-readable code>",
  "instance": "<request URL path>",
  "requestId": "<uuid>"
}
```

For the full error contract and `HttpException` reference, see [Exceptions](/http/exceptions).

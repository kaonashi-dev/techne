---

This page documents the Techne RFC 7807 problem document for HTTP `gateway-timeout` errors.

A `problem+json` response with `type` of
`https://github.com/kaonashi-dev/techne/blob/main/docs/errors/gateway-timeout.md`
indicates the server encountered a Gateway Timeout condition.

```json
{
  "type": "https://github.com/kaonashi-dev/techne/blob/main/docs/errors/gateway-timeout.md",
  "title": "Gateway Timeout",
  "status": 504,
  "detail": "<human-readable explanation>",
  "code": "<optional stable machine-readable code>",
  "instance": "<request URL path>",
  "requestId": "<uuid>"
}
```

For the full error contract and `HttpException` reference, see [Exceptions](/http/exceptions).

---

This page documents the Techne RFC 7807 problem document for HTTP `too-many-requests` errors.

A `problem+json` response with `type` of
`https://github.com/kaonashi-dev/techne/blob/main/docs/errors/too-many-requests.md`
indicates the server encountered a Too Many Requests condition.

```json
{
  "type": "https://github.com/kaonashi-dev/techne/blob/main/docs/errors/too-many-requests.md",
  "title": "Too Many Requests",
  "status": 429,
  "detail": "<human-readable explanation>",
  "code": "<optional stable machine-readable code>",
  "instance": "<request URL path>",
  "requestId": "<uuid>"
}
```

For the full error contract and `HttpException` reference, see [Exceptions](/http/exceptions).

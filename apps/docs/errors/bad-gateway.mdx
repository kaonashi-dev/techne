---

This page documents the Techne RFC 7807 problem document for HTTP `bad-gateway` errors.

A `problem+json` response with `type` of
`https://github.com/kaonashi-dev/techne/blob/main/docs/errors/bad-gateway.md`
indicates the server encountered a Bad Gateway condition.

```json
{
  "type": "https://github.com/kaonashi-dev/techne/blob/main/docs/errors/bad-gateway.md",
  "title": "Bad Gateway",
  "status": 502,
  "detail": "<human-readable explanation>",
  "code": "<optional stable machine-readable code>",
  "instance": "<request URL path>",
  "requestId": "<uuid>"
}
```

For the full error contract and `HttpException` reference, see [Exceptions](/http/exceptions).

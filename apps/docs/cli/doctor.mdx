title: "techne doctor"
description: "Diagnose `tsconfig` / project layout."
---

```bash
# Diagnose tsconfig / project layout
bunx @kaonashi-dev/techne doctor
```

`doctor` performs a series of static checks against the project layout —
TypeScript config sanity, presence of `techne.config.ts`, and similar
ergonomic gotchas. It also detects legacy `bnest.config.ts` filenames and
prints a yellow "deprecated" line telling you to rename — functional
behavior is unchanged. Run it whenever the framework reports a confusing
startup error.

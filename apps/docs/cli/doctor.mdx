```bash
# Diagnose tsconfig / project layout
bunx @kaonashi-dev/techne doctor
```

`doctor` performs a series of static checks against the project layout —
TypeScript config sanity, presence of `techne.config.ts`, and similar
ergonomic gotchas. Run it whenever the framework reports a confusing startup
error.

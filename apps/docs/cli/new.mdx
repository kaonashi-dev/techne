---
title: "techne new"
description: "Scaffold a new Techne project (alias: techne create)."
---

```bash
# Create a new project
bunx @kaonashi-dev/techne new my-project
```

`new` and `create` are aliases. The generated project uses
`@kaonashi-dev/techne/common` and `@kaonashi-dev/techne/core` and emits a
`bootstrap()` `main.ts` paired with a `techne.config.ts` skeleton — see
[Quickstart](/quickstart) for the resulting layout.

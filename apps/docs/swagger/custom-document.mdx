---
title: "Custom OpenAPI emitter"
description: "Lower-level helpers when you need to integrate the OpenAPI emitter directly."
---

The lower-level pieces are exported for callers that want to integrate the
emitter directly: `emitOpenApiDocument(app, builder?)` and
`typeboxToOpenApi(schema)` from `@kaonashi-dev/techne/swagger`.

```ts
import { DocumentBuilder, emitOpenApiDocument, typeboxToOpenApi } from "@kaonashi-dev/techne/swagger";

const document = emitOpenApiDocument(
  app,
  new DocumentBuilder().setTitle("My API").setVersion("1.0.0"),
);

const schema = typeboxToOpenApi(SomeTypeBoxSchema);
```

Use these when you want to post-process the document (e.g. tag filtering,
component injection, security-scheme assembly) before serving or shipping
it to clients. The helpers return plain JSON-compatible objects so the
result is easy to round-trip through `JSON.stringify` or
`Bun.file().write()`.

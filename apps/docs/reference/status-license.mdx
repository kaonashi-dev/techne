## Status

Techne is still experimental. APIs may change quickly, some areas are
incomplete, and documentation will continue to evolve with the framework.

v0.3 (in progress) introduces declarative config, plugin protocol, RFC
7807 errors, `/healthz` + `/readyz`, graceful shutdown, expanded CLI, and
auto OpenAPI.
